# Structure Tools

Tools for analyzing and structuring processed survey data.

## Files

- **`create_structured_tables.py`** - Convert decrypted data to structured analysis-ready formats
- **`analyze_encryption_limits.py`** - Analyze encryption performance and size limits
- **`realistic_location_analysis.py`** - Tools for analyzing location data patterns
- **`encryption_limits_report.md`** - Detailed analysis report on encryption limits

## Quick Start

1. Create structured tables from decrypted data:
   ```bash
   python create_structured_tables.py
   ```

2. Analyze encryption performance:
   ```bash
   python analyze_encryption_limits.py
   ```

3. Analyze location data:
   ```bash
   python realistic_location_analysis.py
   ```

## Output Formats

- **CSV files** - For statistical analysis in R, Python, etc.
- **JSON files** - For web applications and APIs
- **Excel files** - For manual review and sharing
- **Reports** - Markdown and HTML analysis reports

See `../STRUCTURED_DATA_GUIDE.md` for detailed documentation.
